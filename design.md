# Design — Ahmed Khaled Portfolio

Locked design system for recruitment-focused portfolio. Every section shares this system.

## Genre
modern-minimal

## Macrostructure family
- Landing (single page): Split Studio hero, timeline experience, bento projects, compact skill matrix, statement contact

## Theme
- `--color-paper`   oklch(14% 0.02 260)
- `--color-paper-2` oklch(18% 0.025 260)
- `--color-ink`     oklch(96% 0.01 90)
- `--color-ink-2`   oklch(72% 0.02 260)
- `--color-rule`    oklch(28% 0.02 260)
- `--color-accent`  oklch(62% 0.14 265)
- `--color-focus`   oklch(70% 0.16 265)

## Typography
- Display: Fraunces, weight 500
- Body: Plus Jakarta Sans, weight 400–600
- Mono: JetBrains Mono, weight 400–500

## Motion
- Reveal: fade + 12px slide, stagger 80ms
- Reduced motion: opacity-only, ≤150ms

## CTA voice
- Primary: filled accent, rounded-lg, sentence case
- Secondary: hairline border, transparent fill
